一、系统概述
-----------
本系统是一个基于 Flask 的单文件全栈 Web 应用，用于可视化展示 Python 核心概念
的内存模型和运行机制。系统通过 SVG 图形、代码高亮面板、步骤导航、C/C++ 对比
面板和实际运行结果，帮助学习者直观理解 Python 的底层原理。

启动方式: python app.py → 浏览器访问 http://127.0.0.1:5000


二、作业基础要求完成情况
-----------------------

1. ✓ 能够对 C 和 Python 的变量在发生赋值后的内存变化进行可视化展示
   - 第1节"动态类型": 逐步展示 x 从 int→float→str→list→dict 的类型变化
   - 每个步骤包含 SVG 内存图 + 代码高亮 + C++ 对比面板
   - 后台 API 返回实际 id 值，验证变量只是"名字标签"

2. ✓ 能够对列表的内存进行可视化展示，包括二维列表
   - 第2节"列表内存": 展示一维列表的引用关系 (b=a, c=a.copy())
   - 第2D节"二维列表": 展示二维列表的指针链结构
     · 外层列表 → 指针数组 → 内层列表对象的内存布局
     · row0 = matrix[0] 共享同一内层列表
     · 浅拷贝只复制外层、深拷贝递归复制的区别
     · 修改 row0/shallow 对原矩阵的影响
   - 简洁清晰的 SVG 图展示指针跳转和共享/独立关系

3. ✓ 能够可视化展示引用、浅拷贝和深拷贝
   - 第2节: b=a (引用) vs c=a.copy() (浅拷贝) 的区别
   - 第2D节: 二维列表的浅拷贝与深拷贝对比
   - 第4节"深/浅拷贝": 嵌套列表的完整演示
     · .copy() 浅拷贝 → 内层列表共享
     · copy.deepcopy() 深拷贝 → 完全独立
   - 所有拷贝操作都展示实际 id 对比


三、超出作业要求的功能 (额外实现)
---------------------------------

1. 【C/C++ 对比面板】
   每个知识点都包含 C++ vs Python 的并排代码对比和核心差异分析，帮助有 C/C++
   背景的学习者理解 Python 的不同设计哲学。

2. 【函数参数传递可视化】(第3节)
   - 不可变对象 (int) → 类似 C++ 传值
   - 可变对象 (list) → 类似 C++ 传指针
   - 赋值(=) vs 修改(.append) 的关键区别
   - 三种场景的 SVG 图解 + 实际运行验证

3. 【字符串拼接性能分析】(第5节)
   - 使用 Chart.js 柱状图对比 + 拼接 vs join 拼接的性能差距
   - 理论分析: 字符串不可变性导致 O(n²) vs O(n)
   - 后台实时性能测试，返回毫秒级对比数据
   - 包含 C++ std::string 可变性对比

4. 【Sci-Fi 科幻风格 UI】
   - 星空粒子背景 (Canvas 动画 + 星座连线)
   - 光标跟随光晕效果
   - 扫描线叠加层 (HUD 风格)
   - 霓虹发光边框和按钮特效
   - 全息微光卡片悬浮动画
   - 点击粒子爆散效果
   - 自适应网格布局

5. 【交互式步骤导航】
   - 每个知识点分多步展示，支持上一步/下一步/点击跳转
   - 代码高亮面板同步显示当前步骤对应的代码行
   - SVG 内存图随步骤动态变化

6. 【实际运行验证】
   - 每个知识点都有"运行完整演示"按钮
   - 后台实际执行 Python 代码，返回真实的内存 id 值
   - 结果以结构化面板展示，关键信息用颜色标注 (共享=红色, 独立=绿色)

7. 【完整代码折叠区】
    每个知识点包含可折叠的完整 Python 代码，方便学习者对照查看


四、系统架构
-----------
- 后端: Flask 单文件应用，6个 API 路由提供交互式演示数据
- 前端: 原生 HTML/CSS/JS + Chart.js (CDN)，无其他依赖
- 可视化: 内联 SVG 动态生成 + Canvas 背景特效
- 主题: 深色科幻风格，CSS 变量管理配色
- 模块: 动态类型 | 列表内存 | 二维列表 | 函数参数 | 深/浅拷贝 | 字符串性能

五、文件清单
-----------
- app.py     : Flask 单文件全栈应用 (约 2000 行)
