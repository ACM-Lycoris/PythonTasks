【项目概述】
本项目是一个集成"中国行政区划展示"与"遥感研究数据可视化"的
全栈 Web 应用。用户可在地图上从全国视图逐级下钻：
  全国（34 个省）→ 点击省份 → 进入省份视图（显示该省所有地级市）
                  → 点击城市 → 进入城市视图（显示介绍/文化底蕴/景点等）
同时保留原项目的遥感研究数据点功能。

============================================================
【数据集统计】
============================================================
  ★ 省级行政区：34 个（含 4 直辖市、5 自治区、2 特别行政区、台湾省、22 省）
  ★ 地级行政区：343 个（地级市、自治州、盟、地区、省直辖县级市、特别行政区）
  ★ 遥感研究数据点：59 条
  ★ 数据年份范围：2020 - 2023

  每个省份包含字段：
    - 名称、省会、简称、人口、面积、介绍、文化特色

  每个城市包含字段：
    - 名称、所属省份、经纬度、行政级别
    - 城市介绍（约 80-150 字）
    - 文化底蕴
    - 人口（万人）、面积（km²）、GDP（亿元）
    - 气候特点、著名景点、建城历史

============================================================
【作业明确要求之外实现的额外功能】
============================================================

1. ★★★ 三级行政区划下钻交互（核心扩展）
   - 全国视图：地图上以彩色徽章显示 34 个省级行政区
   - 点击省份徽章 → 地图自动飞行放大到该省（flyToBounds 平滑动画）
   - 省份视图：显示该省所有地级市的彩色图标
     · 省会/直辖市：金色五角星 ★
     · 地级市：蓝色圆点 ●
     · 自治州/盟/地区：紫色菱形 ◆
     · 特别行政区：粉色 ✦
   - 点击城市图标 → 弹出详细信息卡片（介绍/文化/人口/GDP/景点等）
   - 同时左侧面板自动同步切换为对应视图

2. ★ 面包屑导航
   - 顶部导航条：🇨🇳 全国 › 陕西省 › 西安市
   - 可点击任意层级返回到对应视图
   - 提示当前所在视图，并显示操作提示

3. ★ 全文搜索框
   - 顶部搜索框，支持城市名称和省份名称模糊搜索
   - 实时下拉显示搜索结果（带防抖优化）
   - 点击搜索结果直接跳转到对应城市（自动先进入省份再聚焦城市）

4. ★ 省份/城市详细信息面板
   - 省份面板：省会、人口、面积、下辖城市数、省份介绍、文化特色、下辖城市清单
   - 城市面板：行政级别、人口、面积、GDP、建城历史、气候、城市介绍、文化底蕴、著名景点
   - 城市列表点击可直接进入该城市详情

5. ★ 全国数据概览统计面板
   - 省级行政区数、地级行政区数、遥感数据点数、年份范围
   - 卡片式布局，鼠标悬停有微动画

6. ★ 快速跳转下拉框
   - 在全国视图侧边栏提供省份下拉框
   - 显示每省的城市数量
   - 选择后直接进入该省视图

7. ★ 图层显示开关
   - 可独立控制"省份标记"和"遥感数据点"是否显示
   - 切换后地图实时更新

8. ★ 多种底图切换（4 种）
   - 高德地图（默认）：标准道路+地名
   - 高德卫星图：高分辨率遥感影像
   - 高德卫星+路网：卫星图叠加地名
   - OpenStreetMap：开源地图

9. ★ 遥感数据点 Marker Cluster 聚合
   - 数据点密集时自动聚合为数字圆形（绿/橙/红三色）
   - 缩放或点击可展开

10. ★ 平滑地图动画
    - flyTo / flyToBounds 实现视图切换的过渡动画
    - 进入省份时自动计算城市范围并调整视野
    - 比例尺、缩放控件等

11. ★ 美观的弹出信息卡片
    - 不同视图使用不同颜色头部（城市绿色、研究点红色）
    - 信息分类显示，带徽章标签
    - 滚动条美化

12. ★ 自定义图标设计
    - 省份徽章使用省份简称（如：京、沪、粤）
    - 城市图标根据行政级别区分颜色和形状
    - 鼠标悬停有缩放动画

13. ★ 响应式布局
    - 桌面（1024px+）：标准三栏布局
    - 平板（768-1024px）：紧凑布局
    - 手机（<768px）：上下分栏布局

14. ★ 侧边栏折叠
    - 折叠按钮可隐藏侧边栏最大化地图区域
    - 自动重新计算地图尺寸

15. ★ 错误处理
    - 网络异常显示 Toast 提示
    - 自动消失，不打断用户操作

【API 端点设计】
============================================================
  GET /                          - 主页
  GET /api/provinces             - 所有省份元数据
  GET /api/province/<name>       - 单个省份详情 + 下属城市列表
  GET /api/cities                - 所有城市（可 ?province=xx 筛选）
  GET /api/city/<id>             - 单个城市完整信息
  GET /api/search?q=xxx          - 城市/省份模糊搜索
  GET /api/points                - 遥感研究数据点（可 ?province=xx&year=xxx）
  GET /api/years                 - 研究年份列表
  GET /api/stats                 - 全局统计数据

【运行步骤】
============================================================
  1. 安装 Python 3.8+
  2. 进入项目目录：cd E:\Python\PythonTask\4
  3. 安装依赖：pip install -r requirements.txt
  4. 初始化数据库：python init_db.py
     （会生成 remote_sensing.db，包含 34 省 + 343 市 + 59 遥感数据点）
  5. 启动服务：python app.py
  6. 浏览器访问：http://127.0.0.1:5000

【使用示例】
============================================================
  操作 1：查看陕西省 → 点击地图上的"陕"徽章
         → 地图飞行放大到陕西省，显示西安、宝鸡、咸阳等 10 个地级市
         → 左侧显示陕西省介绍、文化、下辖城市列表

  操作 2：查看西安市 → 在省份视图点击西安★（金色省会图标）
         → 弹出西安详情，左侧显示完整介绍：
            "西安，陕西省省会，'千年帝都'，十三朝古都，世界四大古都之一，
             丝绸之路起点。汉唐文化、周秦文化、关中文化、秦腔、皮影..."
         → 列出景点：兵马俑、华清池、大雁塔、明城墙、回民街、华山

  操作 3：搜索 → 顶部输入"杭州"
         → 下拉显示"杭州市 - 浙江省·副省级"
         → 点击直接跳转到杭州详情

  操作 4：返回上级 → 点击面包屑或"返回"按钮
         → 城市 → 省份 → 全国

【数据真实性说明】
============================================================
  - 所有省份/城市的人口、面积、GDP 数据均参考权威统计数据
