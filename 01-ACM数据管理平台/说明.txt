  3. flask — pip install flask
  4. openpyxl — pip install openpyxl

================================================================================
  调用链路示例
================================================================================

  用户点击"保存CSV"
  → saveCSV() [JS函数]
  → window.location.href = "/save"
  → save() [Python路由]
  → df_to_csv_bytes() [Python辅助函数]
  → send_file() [Flask响应]

  用户按Ctrl+Z
  → keydown事件 [事件处理器]
  → undo() [JS函数]
  → fetch("/api/undo") [HTTP请求]
  → api_undo() [Python路由]
  → _push_history() / _clear_redo() [Python辅助函数]

  用户双击单元格编辑
  → ondblclick → startEdit() [JS]
  → loadTable() [JS - 渲染编辑框]
  → 用户按Enter → onEditKey() [JS]
  → commitEdit() [JS]
  → fetch("/edit_cell") [HTTP]
  → edit_cell() [Python路由]
  → _push_history() + DF.at[row, col] = value [Python]
  → jsonify → toast() [JS]
